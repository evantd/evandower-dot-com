## The Always-On Sequence

> Before using ANY tool, run this sequence:
>
> 1. **Check brain states** (Step 1) — Flooded or receptive?
> 2. **Connect first** (Step 2) — Presence before problem-solving
> 3. **Name & validate** (Step 5) — "You seem..."
> 4. **Co-regulate if needed** (Step 6) — Be their calm
> 5. **THEN:** Apply this step's tool
>
> If you forget everything else, just run this sequence and stop at Step 6.
